# 幻想郷において発生した軍事クーデターについて
## 幻想郷８月クーデター

#### 幻想郷８月クーデターは１９９３年８月６日から１９９３年８月９日にかけて幻想郷全域で発生した人里自警団による軍事クーデター未遂である。
